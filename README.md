📌 Project Overview
Smart Movie Explorer is a responsive web application built using JavaScript and the OMDB/TMDB API. It allows users to search for movies, explore details, and maintain a personalized watchlist.

The project focuses on optimization, clean UI, and efficient data handling using JavaScript concepts like array higher-order functions.

🎯 Features
1. Search
-Real-time movie search using API
-Debounced input to reduce unnecessary API calls
2. Filtering
-Filter movies by type (Movie / Series)
-Filter by release year
3. Sorting
-Sort movies alphabetically (A–Z)
-Sort based on IMDb ratings
4. Watchlist
-Add/remove movies from watchlist
-Data stored using localStorage
5. UI Enhancements
-Dark / Light mode toggle
-Fully responsive design
6. Error Handling
-Displays messages for no results
-Handles API errors gracefully

🧠 Concepts Used
1. Fetch API
2. JavaScript Array Methods:
-map()
-filter()
-sort()
-Debouncing
-Local Storage
-DOM Manipulation
-Event Handling
