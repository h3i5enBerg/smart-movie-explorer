# 🎬 Movie Explorer

A modern, responsive movie discovery web application that helps users search, filter, and manage movies efficiently using real-time API data. 

## 🚀 Live Demo
[https://smartmovieexplorer.netlify.app/](https://smartmovieexplorer.netlify.app/)

## 📌 Project Overview
**Movie Explorer** is a sleek web application built using JavaScript and movie APIs (TMDB/OMDB). It allows users to search for movies, explore details, and maintain a personalized watchlist.

The project focuses on a clean, stylized UI, smooth interactions matching premium streaming services, and efficient data handling using JavaScript higher-order functions.

## 🎯 Features

1. **Search & Navigation**
   - Real-time movie search using API
   - Debounced input to reduce unnecessary API calls
   - Premium top navigation bar featuring Login and Sign Up actions

2. **Filtering & Sorting**
   - Filter movies by release year and type (Movie / Series)
   - Sort movies alphabetically (A–Z)
   - Sort based on IMDb ratings

3. **Watchlist Management**
   - Add/remove movies from a persistent watchlist
   - Data stored securely using localStorage

4. **UI Enhancements**
   - Seamless Dark / Light mode toggle
   - Fully responsive design built with Flexbox/Grid
   - Clean, modern component styling (pill-shaped searchbar, styled buttons)

5. **Error Handling**
   - Displays clear messages for no results
   - Handles API errors gracefully

## 🧠 Concepts Used
- **Fetch API** for asynchronous requests
- **JavaScript Array Methods**: `map()`, `filter()`, `sort()`
- **Debouncing** techniques for performance
- **Local Storage** for user data persistence
- **DOM Manipulation** & **Event Handling**

## 🛠️ Tech Stack
- HTML5
- CSS3 (Vanilla CSS, custom styling)
- Vanilla JavaScript (ES6+)
- TMDB / OMDB API
